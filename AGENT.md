# Agent Notes

This repository contains a local OpenRouter proxy plus a macOS SwiftUI menu bar monitor for Claude prompt cache visibility.

## Components

- `openrouter-ttl-1h-proxy.mjs`: Node.js proxy that listens on `CLAUDE_CACHE_HOST` / `CLAUDE_CACHE_PORT`, forwards traffic to `OPENROUTER_BASE_URL`, rewrites Opus cache controls to `ttl: "1h"`, tracks sessions, and exposes `GET /__status`.
- `menubar-app/`: Swift Package Manager executable target for the native menu bar app.
- `launchd/`: portable plist templates. Installed plists are generated by `scripts/install-launchd.sh`.
- `test/`: Node.js unit tests for request rewriting and upstream URL mapping.

## Useful Commands

```bash
npm test
npm run check:proxy
npm run build:menubar
npm run check:plist
npm run check
```

Install local LaunchAgents:

```bash
npm run install:launchd
```

Uninstall local LaunchAgents:

```bash
npm run uninstall:launchd
```

## Maintenance Rules

- Do not commit `menubar-app/.build/`, generated LaunchAgent plists, logs, or runtime state.
- Keep user-specific paths out of committed plist templates.
- If the proxy port changes, update launchd environment values and `CLAUDE_CACHE_STATUS_URL` together.
- Keep README usage examples and launchd scripts aligned.
